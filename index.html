What I Do

Frontend-developer

Creating the user interface and interactive aspects of a website or application, using languages like HTML, CSS, and JavaScript to ensure a seamless user experience. They collaborate closely with designers and back-end developers to integrate the front-end with the back-end functionality.

Web Design

Focusing on the visual aesthetics and layout of a website, creating the overall look and feel. They use design software and tools to develop the site's appearance, including color schemes, typography, and graphic elements, to ensure a visually appealing and user-friendly experience.

Ecommerce

Creating an e-commerce website with front-end technology involves designing a visually appealing and user-friendly interface for browsing products, implementing features like search, filters, and product pages using HTML, CSS, and JavaScript, and integrating with back-end systems for functionality like user authentication, shopping cart management, and payment processing.

Marketing

showcasing my skills and projects through a professional portfolio, engaging on platforms like GitHub and LinkedIn to network and sharing my work, and staying updated with industry trends to demonstrate expertise and attract potential clients or employers.
